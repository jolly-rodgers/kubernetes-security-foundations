# SentinelAdmission Roadmap

## Phase 1 (MVP)
- Block privileged containers
- Block hostPath volumes
- Enforce runAsNonRoot
- Clear, actionable deny messages

## Phase 2
- RBAC wildcard detection
- Mandatory workload identity labels
- Pod Security Standards alignment
- CI policy enforcement with Conftest

## Phase 3
- Admission audit event export (JSON)
- Multi-cluster packaging (Helm/Kustomize)
- Centralized policy distribution
